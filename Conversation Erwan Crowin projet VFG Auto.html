📋 Contexte & Problématique

VFG Auto est une plateforme web full-stack développée pour moderniser la gestion d'un garage automobile.

La problématique : Le garage gérait manuellement les rendez-vous et les données clients, rendant difficile l'optimisation du planning des mécaniciens et la traçabilité des interventions.

La solution : Une application centralisée permettant aux clients de réserver facilement leurs rendez-vous en ligne, et offrant aux administrateurs des outils puissants de gestion avec calendrier interactif, statistiques détaillées et suivi complet des véhicules.

🎯 Objectifs du projet

  • Digitaliser le processus de prise de rendez-vous
  • Optimiser la gestion du planning des mécaniciens
  • Centraliser les données clients et véhicules
  • Fournir des statistiques en temps réel
  • Améliorer l'expérience utilisateur client
  • Assurer une sécurité maximale des données

✨ Innovation clé

🚗

Système de réservation intelligent avec sélection de services par bulles visuelles, gestion multi-véhicules, calendrier dynamique avec disponibilités en temps réel, et attribution automatique des mécaniciens selon leurs compétences et disponibilités.

🏗️ Architecture Technique

Base de données MySQL

Architecture de base de données complète avec 13 tables interconnectées :

Tables principales :

  • users - Comptes utilisateurs (clients & admins)
  • rdv - Rendez-vous avec statuts et mécaniciens
  • services - Catalogue services automobiles
  • vehicules - Informations véhicules clients
  • mecaniciens - Profils mécaniciens du garage

Tables support :

  • disponibilites_mecaniciens - Planning hebdomadaire
  • competences_mecaniciens - Compétences/niveaux
  • absences_mecaniciens - Gestion congés
  • horaires_garage - Horaires d'ouverture
  • jours_fermeture - Fermetures exceptionnelles
  • historique_entretien - Historique interventions
  • notifications - Système notifications

Points techniques avancés :

  • ✅ Triggers SQL pour validation automatique des rendez-vous
  • ✅ Contraintes d'intégrité avec CASCADE pour cohérence données
  • ✅ Indexes sur colonnes fréquemment requêtées
  • ✅ Transactions PDO pour garantir cohérence
  • ✅ Requêtes préparées anti-injection SQL

Stack Technologique

Backend :

  • PHP 8.0 - Programmation orientée objet
  • PDO - Abstraction base de données
  • Sessions - Gestion authentification
  • Bcrypt - Hash sécurisé mots de passe

Frontend :

  • Bootstrap 5 - Framework CSS responsive
  • JavaScript ES6 - Interactivité client
  • FullCalendar - Bibliothèque calendrier
  • Material Design Icons - Iconographie

💡 Innovations & Points Forts

🚀 Performance

Code optimisé avec gestion intelligente des requêtes SQL et chargement asynchrone pour une expérience utilisateur fluide.

🔒 Sécurité

Protection complète contre XSS, CSRF et injections SQL. Hash bcrypt et gestion sécurisée des sessions.

Interface intuitive avec feedback visuel immédiat, animations fluides et processus simplifié en 3 étapes.

📊 Analytics

Dashboard avec statistiques détaillées, graphiques interactifs et KPIs pour aide à la décision.
