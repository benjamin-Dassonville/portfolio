Vue d'ensemble

Le "Portail de gestion d'entreprise en ligne" est une application web complète permettant aux entreprises de gérer efficacement leurs activités quotidiennes, notamment la gestion des employés, des projets, et des ressources.

Ce projet a débuté comme un travail de cours, mais ma passion pour le développement d'applications métier m'a poussé à l'améliorer et à le perfectionner bien au-delà des exigences initiales.

L'objectif principal était de créer une solution centralisée et intuitive qui pourrait s'adapter aux besoins spécifiques de différentes entreprises, tout en offrant une sécurité robuste et une expérience utilisateur fluide.

Technologies utilisées

Frontend

HTML5 CSS3 JavaScript Bootstrap

Interface utilisateur responsive et moderne, adaptée à tous les appareils. Utilisation de Bootstrap pour garantir une expérience cohérente et une mise en page fluide.

Backend

PHP MySQL

Architecture MVC (Modèle-Vue-Contrôleur) implémentée en PHP pour une organisation claire du code. Base de données relationnelle MySQL pour stocker et gérer efficacement les données.
